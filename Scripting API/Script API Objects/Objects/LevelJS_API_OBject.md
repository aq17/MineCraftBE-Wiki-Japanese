# LevelJS API Object

|タイプ |名称  |説明 |
|---|---|---|
|String |\_\_type\_\_ |読み取り専用です。オブジェクトの種類を定義しています。「level」になります。|
|Positive Integer |level_id |読み取り専用です。レベルの一意の識別ナンバーを表しています。 |
